After you register — set up the redirect
1

Register the domain

Use any registrar below. Enable auto-renew immediately so it never accidentally lapses.

2

Enable WHOIS privacy

Hide your contact details so attackers can't use your registration info for social engineering.

3

Set a 301 redirect

In the DNS/forwarding settings, redirect the domain → your real site. This also captures any misdirected traffic.

4

Add to SwarmHawk monitoring

Add the registered typosquat to your domain list with re-scan enabled — so you're alerted if someone tries to transfer it.

Pro tip: A 301 redirect from the typosquat domain to your main site not only protects customers — it also passes link equity if anyone accidentally links to the typosquat. Configure it as: typosquat.com → yourdomain.com
