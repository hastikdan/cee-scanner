MCP Integration

SwarmHawk MCP Server

The Model Context Protocol (MCP) is an open standard that lets AI assistants call external tools directly from a conversation. Once you add the SwarmHawk MCP server to your AI client, you can say "scan example.com for security issues" and the assistant will call our API, get live results, and explain them to you — no copy-paste, no context switching.

Self-hosting lets you run the MCP server locally (stdio transport) — useful when you can't use HTTP/SSE endpoints or want to keep traffic internal.

1
Install dependencies
terminal 
pip install fastmcp requests python-dotenv
2
Create swarmhawk_mcp.py
swarmhawk_mcp.py 
import os, requests
from fastmcp import FastMCP

mcp = FastMCP("SwarmHawk")
API_KEY = os.getenv("SWARMHAWK_API_KEY", "")
BASE = "https://swarmhawk-backend.onrender.com"
HDR = {"X-API-Key": API_KEY}

@mcp.tool()
def scan_domain(domain: str) -> dict:
    """Run a full 22-check security scan on any domain."""
    r = requests.post(f"{BASE}/api/v1/scan",
                      json={"domain": domain}, headers=HDR, timeout=60)
    r.raise_for_status()
    return r.json()

@mcp.tool()
def quick_risk_score(domain: str) -> dict:
    """Get just the numeric risk score for a domain (fast)."""
    r = requests.get(f"{BASE}/api/v1/risk/{domain}", headers=HDR, timeout=10)
    r.raise_for_status()
    return r.json()

if __name__ == "__main__":
    mcp.run()
3
Configure Claude Desktop for stdio
claude_desktop_config.json 
{
  "mcpServers": {
    "swarmhawk": {
      "command": "python3",
      "args": ["/path/to/swarmhawk_mcp.py"],
      "env": {
        "SWARMHAWK_API_KEY": "swh_your_key"
      }
    }
  }
}

All 7 MCP Tools

Each tool maps to one or more SwarmHawk API endpoints. The AI assistant picks the right tool based on your prompt automatically.

scan_domain ~30–45s
Full 22-check security scan. Returns risk score, all check results, status breakdown, and AI threat summary. The most complete tool — use for thorough analysis.
domainstringTarget domain, e.g. example.com
quick_risk_score ~3–5s
Returns only the numeric risk score (0–100) and critical/warning counts. Useful for quick checks, batch comparisons, or routing logic in agent workflows.
domainstringTarget domain
check_breach_exposure ~3s
HaveIBeenPwned lookup only. Returns breach count, breach names, and total compromised accounts linked to the domain.
domainstringTarget domain
check_typosquats ~5–8s
Generates 200+ lookalike domain permutations and checks DNS resolution. Returns list of registered typosquat domains with their registration status.
domainstringTarget domain
check_reputation ~8s
Runs threat intelligence checks only: URLhaus, Spamhaus DBL, Google Safe Browsing, VirusTotal. Faster than a full scan when you only need reputation data.
domainstringTarget domain
batch_risk_scores ~10–30s
Scan multiple domains at once and get a ranked risk score table. Ideal for comparing vendors, prospects, or a portfolio of client domains.
domainsstring[]Array of domains, max 10 per call
get_quota instant
Returns your API key's remaining scan quota, plan type, and monthly reset date. Useful for agents that need to budget API calls.
(none)—No parameters needed
